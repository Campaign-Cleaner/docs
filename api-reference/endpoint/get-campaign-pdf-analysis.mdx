---
title: 'Get Campaign PDF Analysis'
openapi: 'POST /v1/get_campaign_pdf_analysis'
---
The **get_campaign_pdf_analysis** API allows you to retrieve the analysis report for any of your Saved Campaigns in [Campaign Cleaner](https://campaigncleaner.com). 

#### <u>Request Body:</u>

_Required:_

**campaign**:

- **id**: The campaign id that will be used in other API calls.
    
#### <u>Response:</u>

Is an Application/pdf File Download.
